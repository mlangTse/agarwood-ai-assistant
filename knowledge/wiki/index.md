# Index

## Concepts

- [[沉香]] - 树脂浸润木材的总入口，解释沉香是什么、如何形成、如何合规表达。
- [[结香]] - 沉香树脂形成机制，包括自然感染、创伤和人工诱导。
- [[沉香树种]] - 沉香属、拟沉香属及关键物种的分类口径。
- [[产区对比]] - 莞香系、惠安系、星洲系及常见导购边界。
- [[莞香系]] - 中国系/莞香系沉香的地域口径、常见香气表达和证据边界。
- [[惠安系]] - 越南、芽庄、柬埔寨、老挝、泰国等惠安系口径的香气和风险边界。
- [[星洲系]] - 马来西亚、印尼、达拉干、马泥涝、加里曼丹等星洲系口径。
- [[产区误区]] - 产区名、沉水、野生、奇楠、老料等市场词的常见误判。
- [[香韵解释]] - 清甜、凉意、奶韵、药感、木质、花蜜、清雅等香韵选项的中文解释。
- [[闻香场景]] - 茶室、书房、静坐、助眠、商务空间、送礼、自用、收藏等使用场景。
- [[熏闻方式]] - 电熏、隔火、线香、随身闻香的差异和适用边界。
- [[气味强度]] - 轻柔、平衡、穿透力强三类强度偏好的导购解释。
- [[工艺保养]] - 香材、线香、香粉、手串和摆件的使用保养边界。
- [[产品形态]] - 香材、手串、香粉、线香、摆件、收藏藏品的选择逻辑。
- [[预算层级]] - 500 入门、3000 进阶、20000 高阶、收藏级的购买建议边界。
- [[购买偏好]] - 新手稳妥、小幅升级、重视稀缺性的导购口径。
- [[价格等级]] - 沉香价格受形态、香韵、来源、结香、检测和合规资料共同影响。
- [[真假鉴别]] - 真伪、来源、检测、复闻和高风险宣传词的判断框架。
- [[CITES]] - 沉香相关物种跨境贸易和附录管理的核心合规入口。
- [[濒危保护]] - 沉香相关树种的资源压力、保护状态和非法采伐风险。
- [[沉香贸易]] - 沉香使用史、经济价值、国际贸易和风险表达。
- [[鉴别风险]] - 真伪、品质、产区、投资和药用宣传中的风险边界。
- [[合规表达]] - 面向用户、导购和文案时的中文表达边界。
- [[资料边界]] - 处理英文来源、重复来源和非沉香资料时的使用规则。
- [[人工种植沉香]] - 以栽培和人工诱导结香为核心的可持续供应口径。
- [[野生沉香]] - 稀缺性、非法采伐和合规风险并存的来源口径。
- [[沉香使用史]] - 沉香在香料、宗教、传统药用和贸易中的历史使用脉络。
- [[闻香体验]] - 处理闻香主观感受、感知研究和英文外围资料的入口。

## Entities

- [[Aquilaria]] - 沉香属，沉香主要来源类群之一。
- [[Gyrinops]] - 拟沉香属，沉香来源类群之一。
- [[Aquilaria sinensis]] - 土沉香/白木香，中国与香港语境中的关键树种。
- [[Aquilaria malaccensis]] - 沉香贸易中的重要树种，保护压力显著。
- [[Aquilaria crassna]] - 东南亚沉香来源物种，资料强调濒危和树脂形成。
- [[土沉香]] - A. sinensis 的中文常用口径，连接白木香、香港沉香树和保护议题。
- [[香港沉香树]] - 与 A. sinensis、非法采伐和本地保护相关的文化生态实体。
- [[Kew POWO]] - 核对 Aquilaria/Gyrinops 分类、异名和接受名的植物学来源。
- [[IUCN]] - 核对沉香相关物种保护状态的来源。
- [[沉香属]] - Aquilaria 的中文别名页，指向沉香属分类和物种资料。
